## 🎬 推奨5カット構成（ストーリー形式）

### Cut 1：ホーム＝価値の提示（最重要）
**画面**：`/home` の「今日のダッシュボード」がフルに見える状態

**キャッチ**：
> **毎日1%が、1年で37倍になる。**

**構成要素**：
- Sho挨拶カード「おはよう！」🔥42日連続
- 4マスダッシュボード（ジャーナル☀️✅🌙✅、ToDo 2/3、習慣 3/5、複利+51%）
- 今日やることカード3件（2つチェック済）

### Cut 2：朝ジャーナル＝毎朝の起点
**画面**：`/journal` 朝モード・記入中

**キャッチ**：
> **5分で、今日が動き出す。**

**構成要素**：
- mood 5段階（😊選択中）
- 睡眠時間入力（7.5h）
- 「今日の一言目標：10分だけ読書する」
- ToDo選択UI（3つ選択済）
- 大きな「☀️ 朝ジャーナルを投稿」ボタン

### Cut 3：複利グラフ＝独自機能
**画面**：`/growth` の複利曲線カード

**キャッチ**：
> **見えない1%を、グラフで見る。**

**構成要素**：
- 「あなたの複利」ガラスカード
- 今の成長 +51% / 理想ペース +51%
- 理想(オレンジ破線) vs 実績(ミント実線) の重畳グラフ
- 30日/90日/1年の予測マイルストーン

### Cut 4：ビジョン＝逆算思考
**画面**：`/vision` の4階層表示

**キャッチ**：
> **夢から逆算、今日の一歩まで。**

**構成要素**：
- 最終ゴール：「活躍できていない人を引き上げる社会を作る」12%
- 3年後：「Rizup月商500万円」28%
- 1年後：「有料ユーザー500人達成」45%
- 今月：「月商20万円」65%（進捗バー目立たせる）

### Cut 5：夜ジャーナル＝数字で実感
**画面**：`/journal` 夜モード・複利スコア表示

**キャッチ**：
> **毎日の複利を、数字で実感。**

**構成要素**：
- 今日の複利スコア 71/100 のバー
- 今朝の目標達成「✅できた」選択
- 感謝3つ入力
- 「🌙 夜ジャーナルを投稿」ボタン

## 🎨 デザインフレームのテンプレ

各スクショは**フレーム込み**で作る（Figma / Canva推奨）：

```
┌───────────────────────────────────────┐
│                                       │ ← 上部 約200px 余白
│   毎日1%が、1年で37倍になる。          │ ← キャッチ（Bold, 64pt）
│   Rizup – 複利成長アプリ              │ ← サブ（24pt）
│                                       │
├───────────────────────────────────────┤
│                                       │
│                                       │
│          [実機スクショ]                │ ← デバイス枠込み
│          (iPhone frame)               │
│                                       │
│                                       │
├───────────────────────────────────────┤
│   🌿 rizup-app.vercel.app             │ ← URL（18pt）
└───────────────────────────────────────┘
```

- **背景**：ミントグリーン `#6ecbb0` → 白へのグラデ、または純白 `#fafafa`
- **フォント**：M PLUS Rounded 1c（アプリと統一）
- **ロゴ**：左上に Rizup（40pt、ミント色）

## 🛠 撮影手順

### Step 1：デモデータ投入（初回のみ）
`public/screenshots/DEMO_DATA.md` の SQL を Supabase で実行してデモユーザー「デモ」を作成。

### Step 2：実機またはシミュレータで撮影
- **実機**：iPhone 15 Pro Max を使用、ホームボタン+電源ボタン同時押し
- **シミュレータ**（Mac 有り）：Xcode → Simulator → iPhone 15 Pro Max を起動し ⌘+S でPNG保存
- **Chrome DevTools**（Mac不要・緊急時）：F12 → Device Mode → iPhone 15 Pro Max → スクリーンショット

### Step 3：ブラウザでログイン
```
https://rizup-app.vercel.app/login
 shohei-demo@example.com
 （DEMO_DATA.md に記載のパスワード）
```

### Step 4：各画面に遷移して撮影
撮影前に：
- ステータスバーを整理（通知バッジなし、電池100%）
- 通信ON、時計を 9:41 に（iOSの定番）
- ダークモードはOFF（ライトモード基準）

### Step 5：Figma で5フレーム組み込み
- [Apple Marketing Resources](https://developer.apple.com/design/resources/) から iPhone 15 Pro Max テンプレをDL
- Figma に貼り付けて、実機スクショを中に配置
- 上部にキャッチコピー、下部にロゴ+URL

### Step 6：書き出し
Figma → Export → PNG 1x → **1290×2796** を確認して保存

## ✅ Apple Review チェックリスト

App Store申請前に確認：

- [ ] スクリーンショットが**最新ビルド**の表示と一致している
- [ ] "iPhone"、"App Store"の文字を含んでいない
- [ ] 他社ロゴ（Facebook・Google等）を含んでいない
- [ ] 実機と比べて**誇張表現**がない（数字の改ざんは即リジェクト）
- [ ] 年齢制限レーティングに沿った内容（4+であれば性的要素なし）
