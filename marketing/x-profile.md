## 7. 翔平さんの言葉の特徴（Reiの観察）

磨き込みで意識したこと：

1. **「でも」の使い方**：完璧そうな話の途中で「でも」を入れて人間味を出す
2. **数字の扱い**：必ず前後の比較（Before/After）を出す
3. **部下の話**：具体的な一人を思い浮かべて書く（抽象化しすぎない）
4. **AIへの依存を隠さない**：Claude Code / Rizupアプリ自体が武器
5. **「僕」と「私」の使い分け**：Xは「僕」、ビジネス文書は「私」
