- ❌ アドバイス調（「〇〇すべき」「〇〇したほうがいい」）

- ✅ 具体的な数字（%・件数・日数）
- ✅ 具体的なエピソード（誰の・何を・いつ）
- ✅ 弱さの開示（迷った・固まった・諦めかけた）
- ✅ 結論より「気づき」

## 7. 代表の言葉の特徴（Reiの観察）

磨き込みで意識したこと：

1. **「でも」の使い方**：完璧そうな話の途中で「でも」を入れて人間味を出す
2. **数字の扱い**：必ず前後の比較（Before/After）を出す
3. **部下の話**：具体的な一人を思い浮かべて書く（抽象化しすぎない）
4. **AIへの依存を隠さない**：Claude Code / Rizupアプリ自体が武器
5. **「僕」と「私」の使い分け**：Xは「僕」、ビジネス文書は「私」

### 8-C. コンパウンドエフェクト・複利（5本）

#### C-01 37倍の法則

```
ダーレン・ハーディの『The Compound Effect』
を読んだ時、衝撃だった。

毎日1%改善 × 365日 = 37倍
毎日1%劣化 × 365日 = 0.025倍（ほぼゼロ）

同じ1%でも、
積み重ねる方向で人生が分かれる。

今日の1%、どっち向き？
```

#### C-02 見えない変化

```
複利のむずかしさは、
最初の3ヶ月、ほぼ変化が見えないこと。

グラフは前半ペタンコで、
後半に急に跳ね上がる。

ほとんどの人は、
ペタンコ期間に諦める。

だから、見える化した。
Rizupの複利グラフは、
小さな1%も全部可視化する。
```

#### C-03 習慣の方程式

```
習慣化の方程式、これだけ覚えてほしい：

継続 = 意思力 ÷ 摩擦

摩擦：「やる前にスマホ触る」「準備が面倒」

減らし方：
・時間を固定（毎朝6:30）
・場所を固定（同じ机）
・道具を固定（同じノート）

意思力を使わない仕組みを作る。
これが複利の起点。
```

#### C-04 方向性の重要さ

```
複利の一番怖いところ：
方向を間違えると、指数関数的に遠ざかる。

だから、Rizupに「ビジョン」と
「アンチビジョン」の両方を入れた。

なりたい方向 × 避けたい方向 = 行動の精度

精度高い1%を、365日積む。
それだけで、未来は変わる。
```

#### C-05 最初の一歩が一番難しい

```
複利の一番大事な数字は、
「1日目」だと思う。

1日目にやった人だけが、
100日目・365日目にたどり着く。

0を1にする、これが一番むずかしい。
1を100にするのは、実は簡単。

今日、0を1にしませんか？
```
