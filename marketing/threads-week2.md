## 投稿1：数字＋弱さ開示フォーマット

**元ネタ（概念）**：「年収◯◯円の僕が今も毎朝やっていること」系
**バズ要因**：具体数字 × 地味な習慣 × 意外性

```
全国1位の店長から起業して半年。
僕が今も毎朝やってる1%の習慣。

・紙に「今日やる1つ」を書く
・Rizup で気分を5段階で記録する
・15分だけ読書する
・冷水で顔を洗う

全部、5分で終わる。
派手じゃないけど、これで1年で複利37倍になる計算。

→ https://rizup-app.vercel.app
```

## 投稿2：対比フォーマット

**元ネタ**：「成長できる人 vs 止まる人」系
**バズ要因**：明確な対比 × 読者が自分を投影できる

```
止まる人は：完璧を目指して動けなくなる
進む人は：1%で満足して、毎日続ける

止まる人は：目標を頭の中だけで持つ
進む人は：紙に書いて、毎日見る

止まる人は：他人と比べる
進む人は：昨日の自分と比べる

僕は1年止まってた側。
だから分かる。
Rizup で「1%の継続」を可視化したら
人生が変わった。
```
**タグ**：#自己成長 #マインドセット
