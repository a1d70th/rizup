## カテゴリ②「毎日前向きになる思考法」（5本）

### Post 6
```
自信がない人に伝えたいこと。

自信は「能力」じゃなくて「経験の数」で決まる。

すごい人が自信あるんじゃなくて、
小さいことをたくさんやった人が自信ある。

だから今日、なんでもいいから1個だけやろう。
腕立て1回でもいい。
本を1ページ読むだけでもいい。

それが明日の自信になる。
```

### Post 7
```
「頑張ってるのに結果が出ない」

それ、頑張りが足りないんじゃなくて
「見てくれる人」がいないだけかもしれない。

人は「誰にも気づかれない努力」を
続けるのがいちばん辛い。

だから、応援してくれる仲間を見つけよう。
「今日これやった」を言える場所を持とう。

それだけで、全然違うから。
```

### Post 8
```
比べるのをやめたら人生変わった、は嘘。

正確には「比べる相手を変えた」。

他人と比べるのをやめて、
昨日の自分と比べるようにした。

「昨日よりちょっとマシ」
これだけでいい。

毎日ちょっとずつ。
気づいたら、半年前の自分と全然違う人になってる。
```

### Post 9
```
20代で挫折した人は、30代で化ける。

これ、大手アパレルで数百人見てきた実感。

20代で順調だった人は、
30代で初めて壁にぶつかったとき折れやすい。

でも20代で挫折を経験した人は、
「あの時より楽」って思えるから粘れる。

今つらい人、それ「将来の武器」を作ってる最中。
```

### Post 10
```
「やりたいことが見つからない」

これ、考えて見つかるものじゃない。
やってみて「あ、これかも」って気づくもの。

だから正解は
「やりたいことを探す」じゃなくて
「目の前のことを全力でやってみる」。

合わなかったら次に行けばいい。
それは失敗じゃなくて「消去法の成功」。
```
