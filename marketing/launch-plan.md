# Rizup ローンチマーケティング計画

---

## 1. アプリローンチまでのタイムライン

```
Week 1-2（4月）：認知フェーズ
  ├ X 毎日投稿開始（Shoの物語を発信）
  ├ YouTube Ep.1-2 公開（共感で掴む）
  └ LP 公開済み → メール登録開始

Week 3-4（4月後半）：期待フェーズ
  ├ YouTube Ep.3-4 公開（AI×起業の話）
  ├ アプリモックアップを X で公開
  ├ 「事前登録」フォームをLPに追加
  └ X でβテスター募集開始

Week 5-8（5月）：開発 & ティザーフェーズ
  ├ 開発の裏側を X / YouTube で実況
  ├ スクリーンショットを小出しに公開
  ├ βテスター10名を選定・招待
  └ YouTube Ep.5-8 公開

Week 9-10（6月前半）：βテストフェーズ
  ├ βテスター10名に限定公開
  ├ フィードバック収集 → 改善
  ├ テスターの声を X で紹介
  └ YouTube「アプリを作って10人に使ってもらった結果」

Week 11-12（6月後半）：正式ローンチ
  ├ アプリ正式公開
  ├ YouTube ローンチ動画
  ├ X で一斉告知（協力者にRT依頼）
  └ 最初の100人を集中的に獲得
```

## 2. YouTube → X → LP → アプリの連携導線

```
【認知】YouTube で Sho の物語を知る
    ↓ 概要欄リンク
【興味】X をフォローして毎日の発信を見る
    ↓ プロフィールリンク
【検討】LP（rizup.vercel.app）を見る
    ↓ メール登録
【行動】アプリをダウンロード・初投稿
    ↓ 継続利用
【課金】Pro / Premium に転換
    ↓ 満足
【紹介】友人に口コミで紹介
```

### 各プラットフォームの役割

| PF | 役割 | KPI |
|---|---|---|
| YouTube | 信頼構築・深い共感・新規流入 | 登録者数・平均再生数 |
| X | 毎日の接点・認知拡大・共感獲得 | フォロワー数・エンゲージメント率 |
| LP | 情報整理・メール獲得・アプリへの導線 | PV・メール登録数 |
| アプリ | メインの居場所・課金・リテンション | DAU・有料転換率・チャーン |

## 3. 最初の100人を集める方法

### Phase A：0→10人（個人的なつながり）

| # | アクション | 期待人数 |
|---|---|---|
| 1 | 友人・知人にDMで個別招待 | 5人 |
| 2 | X で「βテスター募集」投稿 → 応募者を選定 | 3人 |
| 3 | YouTube コメント欄から興味ある人を招待 | 2人 |

**ポイント：** 最初の10人は「文化を作る人」。数より質。

### Phase B：10→30人（コンテンツドリブン）

| # | アクション | 期待人数 |
|---|---|---|
| 1 | YouTube Ep.7「前向きSNSを作る」で告知 | 8人 |
| 2 | X で毎日の開発進捗を発信 → 興味者がLP経由で登録 | 7人 |
| 3 | 既存メンバーの口コミ（1人が1人紹介） | 5人 |

### Phase C：30→100人（仕組みドリブン）

| # | アクション | 期待人数 |
|---|---|---|
| 1 | YouTube ローンチ動画 → 概要欄から登録 | 25人 |
| 2 | X で「ローンチ記念・100人限定無料」キャンペーン | 20人 |
| 3 | メンバーの成長ストーリーを X で紹介 → 共感層が流入 | 15人 |
| 4 | note で「Rizup を作った話」記事 → 検索流入 | 10人 |

## 4. Sho キャラクターを使ったコンテンツ戦略

### Sho の設定

| 項目 | 内容 |
|---|---|
| 名前 | Sho |
| 見た目 | 丸くてかわいい。ミントグリーン #6ecbb0 |
| 性格 | 不完全で親しみやすい。完璧じゃないから信頼できる |
| 口調 | タメ口。友達のように。でも温かい |
| 役割 | アプリ内のAIコーチ × コンテンツのナビゲーター |

### Sho を活用するコンテンツ

| コンテンツ | 使い方 |
|---|---|
| X 投稿 | 「Sho の一言」シリーズ。毎朝の励ましを X にも投稿 |
| YouTube サムネ | Sho のイラストをサムネに配置。統一感 |
| LP | Sho が語りかけるUI。「一緒に始めよう」 |
| アプリ内 | AIフィードバック・朝通知・エラー画面すべて Sho が話す |
| グッズ（将来） | Sho ステッカー・Tシャツ。ブランド強化 |

### X 投稿シリーズ「Sho の一言」

```
【Shoの一言 #1】
完璧な1日じゃなくていい。
「今日も起きて、ここに来た」
それだけですでにすごいことだよ。

【Shoの一言 #2】
「自分なんか」って思う日があっていい。
でも、それを口に出せる場所があるかどうかで
その後の1日が全然変わるんだよ。

【Shoの一言 #3】
3ヶ月前の自分に会えたら
きっと驚くと思う。
「え、こんなに変わったの？」って。
```
