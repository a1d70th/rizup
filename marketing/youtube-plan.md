## 1. チャンネルコンセプト

### ポジショニング

```
      エンタメ
        ↑
  ヒカル  │  
        │  
 ────────┼──────── 
  ビジネス│  自己啓発
  系YouTuber│  
        │  ★ Rizup
        │  （リアル × 共感 × AI）
        ↓
      リアル
```

### 差別化ポイント

| 既存YouTuber | Rizup |
|---|---|
| 成功した後に語る | 成功する前のリアルを晒す |
| 「こうすればうまくいく」 | 「今まさに試してる。結果はまだ分からない」 |
| 上から教える | 同じ目線で一緒に歩く |
| 編集バリバリ | 素朴で親しみやすい |

### ターゲット視聴者

- 20〜30代の男女
- 今の仕事や生活に「これでいいのかな」と思っている人
- 起業・副業に興味はあるが一歩踏み出せない人
- AIに興味があるが何から始めればいいか分からない人

## 3. 「無職になった理由」から始まるストーリー設計

### 全体のナラティブアーク

```
起：挫折（Ep.1-3）
  「全国1位だったのに、辞めた。今は無職。」
  → 視聴者の共感を獲得

承：挑戦（Ep.4-6）
  「AIで会社を作り始めた。資金ゼロで。」
  → 視聴者の好奇心を刺激

転：仲間（Ep.7-9）
  「一人じゃ無理だった。仲間が必要だった。」
  → コミュニティへの興味を喚起

結：結果（Ep.10）
  「1ヶ月やってみた結果がこれ。」
  → 次のシーズンへの期待
```

### 代表のキャラクター設計

| 要素 | 内容 |
|---|---|
| 共感ポイント | 自信がなかった過去。無職の不安。一人で抱え込んだ経験 |
| 信頼ポイント | 大手アパレル全国1位。離職率-95%。TOEIC 850。広告黒字化 |
| 親しみポイント | 28歳。等身大。失敗も隠さない。カッコつけない |
| 憧れポイント | AIで会社を作る挑戦。自分の力で道を切り開く姿 |
