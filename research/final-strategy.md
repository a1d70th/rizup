# Rizup 最終ビジネス戦略

---

## 1. ビジネスモデル整理

### 収益構造

```
株式会社 Rizup 収益チャネル

├── アプリ課金（メイン・長期）
│   ├ 無料 ¥0     → ユーザー獲得・リテンション
│   ├ Pro ¥980/月  → AIフィードバック・分析・チャレンジ
│   └ Premium ¥2,980/月 → PDF・成長の手紙・Sho限定
│
├── クラウドワークス案件（短期キャッシュフロー）
│   ├ LP制作 ¥3〜8万/件
│   ├ SEO記事 ¥1〜3万/本
│   └ 広告運用 ¥5〜10万/月
│
└── コンテンツ収益（中期）
    ├ YouTube 広告収入
    └ アフィリエイト
```

### ユニットエコノミクス

| 指標 | 値 |
|---|---|
| CAC（顧客獲得コスト） | ¥0（SNS・YouTube・口コミで獲得） |
| ARPU（ユーザー平均単価） | ¥220/月（有料転換率15%・Pro:Premium比 2:1想定） |
| LTV（生涯価値） | ¥2,640（平均継続12ヶ月） |
| チャーンレート目標 | 月8%以下 |
| 粗利率 | 90%以上（原価はAPI費用のみ） |

## 2. 競合との差別化

| 競合 | 彼らの強み | Rizupの差別化 |
|---|---|---|
| **Finch** | ゲーミフィケーション×セルフケア | Rizupは「仲間の存在」が核。一人じゃなく、みんなで |
| **Daylio** | シンプルな気分記録 | Rizupは記録に「AI分析」と「人の反応」を加える |
| **X / Instagram** | 圧倒的ユーザー数 | Rizupは「ポジティブしか存在しない」空間。比較・批判ゼロ |
| **note** | 長文コンテンツ | Rizupは「毎日500字」の気軽さ。書くハードルが圧倒的に低い |
| **オンラインサロン** | 有名人のコミュニティ | Rizupは「同じ目線の仲間」が主役。上下関係なし |

### Rizup だけの独自価値（コア・バリュー）

1. **強制ポジティブ設計** — ネガティブリアクション・批判が構造的に存在しない
2. **Sho というキャラクター** — AI × 人間の中間。親しみやすく、毎日そばにいる存在
3. **成長の可視化** — 3ヶ月前の自分と比較。「変わった」を実感できる仕組み
4. **1日2投稿リズム** — 朝（宣言）夜（振り返り）で自然にPDCAが回る

## 5. リスクと対策

| リスク | 発生確率 | 対策 |
|---|---|---|
| ユーザーが集まらない | 中 | YouTube→Xの導線を強化。100人まではRizupが1人1人DM |
| 有料転換率が低い | 中 | 無料で十分な価値→Pro限定の「AIフィードバック」体験で差別化 |
| チャーンが高い | 中 | ストリーク・バッジ・週チャレンジで継続動機。成長の手紙で離脱防止 |
| ネガティブ投稿が増える | 低 | モデレーションAIで投稿前スキャン。段階ペナルティ |
| 競合の模倣 | 低 | Rizupキャラクターとコミュニティ文化は模倣困難 |
| Claude API 費用増大 | 低 | Haiku使用（安価）。Pro/Premium限定で利用をゲート |
