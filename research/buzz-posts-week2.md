## 参考投稿5本（構造の抽出・Rizup 版リライト）

### 1. 数字開示型

**参考構造**：「年収◯◯の僕が毎朝やってること」
```
全国1位の店長から起業して半年。
僕が今も毎朝やってる1%の習慣。

・紙に「今日やる1つ」を書く
・Rizup で気分を5段階で記録する
・15分だけ読書する
・冷水で顔を洗う

全部5分で終わる。
でもこれで1年37倍の複利になる計算。

→ https://rizup-app.vercel.app
```

### 2. 弱さ開示型

**参考構造**：「広告費30万溶かした話」
```
独立1ヶ月目、広告費30万を
3日で溶かしました。

理由：成果計測の設定ミス。
ROAS 78%（赤字）で血の気が引いた。

そこから3ヶ月で ROAS 180% に。
コツは1つ：「毎日1%ずつ改善する仕組み」を作る。

この仕組みの個人版が Rizup です。
→ https://rizup-app.vercel.app
```

### 3. 対比型

**参考構造**：「伸びる人 vs 止まる人」
```
止まる人は：完璧を目指して動けなくなる
進む人は：1%で満足して、毎日続ける

止まる人は：目標を頭の中だけで持つ
進む人は：紙に書いて、毎日見る

止まる人は：他人と比べる
進む人は：昨日の自分と比べる

僕は1年止まってた側。
だから作った。Rizup。
```

### 4. 問いかけ型

**参考構造**：「もし◯◯なら？」
```
1年後、今と同じ毎日を続けてたら
あなたはどうなってる？

僕は大手アパレル店長を辞めた時、
これを自分に問いました。

答えは「変わってない」でした。

だから辞めて、Rizup を作った。
毎日1%の複利で、1年で37倍の成長を可視化するアプリ。

あなたの答えは？
```

### 5. リスト化型

**参考構造**：「やめて良かった5つ」
```
今年やめて良かった5つ：

1. 寝る前のSNS（眠りが変わる）
2. 目標を「大きく」立てる（小さく刻む）
3. 朝のコーヒー（水と白湯に）
4. 他人との比較（昨日の自分だけ見る）
5. 完璧を目指す（1%でOKと決める）

全部 Rizup で可視化できるようにした。
1%の複利は、365日で37倍になる🌿
```

## 共通するバズ要素

1. **冒頭3秒で"具体数字"または"断定"** — スクロール停止率UP
2. **自分の失敗/弱さの開示** — 信頼・共感
3. **「あなた」への問い** — リプ・保存誘発
4. **短文 × 改行多め** — スマホ縦視線に最適化
5. **末尾にリンク or 問い** — CTA
