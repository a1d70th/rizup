## 1. 2025年 習慣化/ジャーナリング系アプリ TOP10 と「なぜコア機能に絞ったか」

| # | アプリ | コア機能 | 削ぎ落としの理由 |
|---|---|---|---|
| 1 | **Streaks** | ToDo に "何連続" だけを持たせる | 習慣化で人を動かすのは数値ではなく「途切れ恐怖」。カレンダー＋連続数だけで十分。 |
| 2 | **Finch** | ペット育成 × 自己ケア1日1-3タスク | ゲーム性を足しつつ、タスク量は3つ上限。過多は離脱要因。 |
| 3 | **Bearable** | 気分/症状/睡眠の"1画面トラッキング" | 健康系は「毎日入れる負荷」が命。入力画面を1つに集約。 |
| 4 | **Reflectly** | 質問1問＋気分5段階 | 書く量を減らすと継続率が上がるという研究（Harvard Health 2022）を採用。 |
| 5 | **Daylio** | タップだけで気分＋アクティビティ記録 | 文字入力ゼロでも日記が成立する＝最低摩擦設計。 |
| 6 | **Habitica** | 習慣＝RPG。ただし最新版は"シンプル派"オプション追加 | 初期はゲーミフィケーション満載→過剰で離脱→軽量モード追加で復活。 |
| 7 | **Fabulous** | "Morning Ritual" 1コース単発 | コンテンツを増やすほど到達率が下がるため、1時間のコース1つに集中。 |
| 8 | **Stoic** | 朝夜ジャーナル＋1哲学の引用 | 機能は朝夜2回だけ。これが継続の秘訣。 |
| 9 | **Oak / Balance** | 瞑想1タップ開始 | 選択肢=摩擦。デフォルト1本を大きく表示。 |
| 10 | **One Sec** | SNSを開く前に1秒遅延させるだけ | 機能が"1つだけ"。だから毎日効く。 |

### 共通点
1. **1画面で1つの意思決定**しか求めない
2. **入力の選択肢を減らす**（タップ/スライダー中心、テキスト任意）
3. **続けることに対する報酬**は「連続数」か「静かな完了表示」に絞る
4. **通知とアニメーションは最小限**（過剰な演出は疲労を生む）

## 2. 「引き算で成功したアプリ」の事例

### Instagram（2010年）
初期はチェックイン/フィルター/共有の複合アプリ "Burbn"。**写真＋フィルターだけに絞った**瞬間に爆発。
→ **学び：複数機能の束より、1つの鋭いユースケース**

### Superhuman（2017年）
メールクライアントから機能を削りに削り、"Inbox Zero を毎日達成する"だけに特化。
→ **学び：ゴールを1つに据える。日課化しやすくなる**

### Day One → Streaks Workout
日記アプリの機能肥大で離脱 → ワークアウト記録だけの別アプリに切り出し、MAU が別アプリより伸びた。
→ **学び：機能を切り出して別アプリにする勇気**

### Notion → Simple Note系の台頭
リッチすぎて毎日は開かない人向けに Bear・Obsidian Daily・Craft が伸びた。
→ **学び：パワーユーザー機能は裏に隠し、毎日の入口はシンプルに**

### ユーザーが本当に毎日使う機能の共通型
- **30秒以内に完了する主要アクション**
- **連続性の可視化**（ストリーク・ヒートマップ）
- **他人の気配**（静かな共感、派手なSNSではない）
- **今日やるべき"1つ"の提示**

## 3. Rizup に残すべきコア機能（提案）

### 残す（MUST）
| 機能 | 根拠 |
|---|---|
| 朝夜ジャーナル | Stoic/Reflectly の成功パターン。1日2タップで継続可能 |
| 今日の一言目標（朝の中に内包） | Fabulous の "Morning Ritual"。単独ページは不要 |
| 習慣トラッカー | Streaks のコア機能。ビジョンからの逆算で意味が増幅 |
| ビジョン（4階層） | Rizup の差別化。最終ゴール→1年→今月で逆算が効く |
| タイムライン（みんなの今日） | Finch 的な"他人の気配"。タブは不要、1本フィード |
| 成長グラフ（複利曲線） | Rizup の核。ここが数字で"効いてる実感"を出す |
| プロフィール/設定 | PWA に必須 |

### 削る（REMOVE）
| 機能 | 削る理由 |
|---|---|
| /today（ToDo 単独ページ） | 朝ジャーナルの「今日の目標」で代替可能。重複 |
| タイムライン 4タブ（全員/フォロー中/朝/夜） | X/Threads は1本フィードが最強。選択肢=摩擦 |
| アンチビジョン手動追加 | 自動生成で十分。入力画面は不要 |
| おすすめ（Recommend） | 毎日のコアループに乗らない。将来 v5 で復活 |
| 紙吹雪/Confetti | 演出過多。連続数の静かな増加で十分 |
| ステータスバーの ToDo 数 | 朝ジャーナル内で完結するので重複 |
| Rizupタイプ診断（プロフィール内） | 月1未満のアクション。初回のみでOK |

### 残すが隠す（HIDE）
- 通知（デフォルトOFF、設定で有効化）
- 画像添付（2タップ奥）
- ダーク/ライト切替（3択ではなく OS 同期デフォルト）

**Rizup のコアループは "朝→習慣→夜→グラフ" の4動作で完結する**。それ以外は削る。
