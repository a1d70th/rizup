# Competitor UI インスピレーション（iOS参考）

> 作成：Kai（CMO）/ 2026-04-15
> 目的：Streaks / Finch / Daylio の UI を観察し Rizup に取り入れる

---

## 1. Streaks（iOS）ホーム画面の特徴

- **円リング12個グリッド**：1日の12個までの習慣を円リングで表示。中身は絵文字のみ。
- **タップで完了**：タップするとリングがグラデで埋まり、中央に✓。
- **タブなし**：ホーム1枚に集約。設定は左上歯車のみ。
- **色は習慣ごと**：視認性UP。
- **文字はほぼない**：絵文字と数字だけで意味が伝わる設計。

→ **Rizup 取り込み**：3リング化は実装済（v4.5）。さらに"色分け"と"中央ダブルタップで完了"の導入検討。

## 2. Finch（iOS）タイムライン画面の特徴

- **ペット中心**：上半分がペットのアバター、下半分がフレンドのアクティビティ。
- **ポジティブのみ**：応援・わかる系のリアクションだけ。ネガ削除。
- **1本フィード**：タブなし。フィルタは上部のセグメントコントロール1列。
- **カード間に線なし**：余白で区切る、カードの影は最小。
- **絵文字プロフィール**：アバターは絵文字1個。可愛さ優先。

→ **Rizup 取り込み**：タイムライン1本化は v4.3 実装済。カードの余白と区切り線を v4.6 で調整済（`divide-y` で統一）。

## 4. Bearable（iOS）ホーム画面の特徴

- **1画面に1日の全データ**：気分・睡眠・症状・習慣がすべて同じ画面。
- **タイル式レイアウト**：2列グリッドで色分け。タップで詳細。
- **スクロールなし**：全てを折りたたまず、視線を上下に動かすだけで完結。
- **グラフと入力が隣接**：過去のトレンドを見ながら今日を記録。

→ **Rizup 取り込み**：成長グラフとジャーナル入力を1タブにまとめる検討（v5）。

## 3. Daylio（iOS）ジャーナル入力の特徴

- **3タップで完了**：気分5段階 → アクティビティタグ → メモ（任意）。
- **キー入力は最後**：最初は選ぶだけ。
- **過去との比較を即表示**：入力後「先週より+2」などの差分表示。
- **月別カレンダー**：毎日の気分を色ドットで可視化。

→ **Rizup 取り込み**：「もっと書く▼」で選ぶだけで完結できる設計は v4.6 実装済。
