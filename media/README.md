## 事業内容

### note（コンテンツ販売）

| コンテンツ種別 | 内容 | 価格帯 |
|---|---|---|
| 無料記事 | 共感・悩み系。SNSからの流入を狙う | 無料 |
| 有料記事 | 具体的な解決策・ノウハウ | 300〜980円 |
| 有料マガジン | 連載形式・まとめ販売 | 1,980円〜 |

**note 記事テーマ例**
- 「自信がない人が一歩踏み出すための3つの習慣」
- 「比べるのをやめたら、人生が変わった話」
- 「やりたいことが見つからない人へ」

### SNS 運用方針

| プラットフォーム | 投稿頻度 | 目的 |
|---|---|---|
| X（Twitter） | 毎日1〜2投稿 | 認知拡大・共感獲得 |
| note | 週1〜2本 | SEO・収益 |
| Instagram | 週3投稿 | ビジュアル訴求・女性層へのリーチ |

**SNS 投稿の3原則**
1. 「わかる」と思ってもらえる共感ポスト
2. 「知らなかった」と思ってもらえる情報ポスト
3. 「行動しよう」と思ってもらえるアクションポスト
