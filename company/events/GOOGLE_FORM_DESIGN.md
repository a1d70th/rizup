# 月 1 オンラインミーティング 参加フォーム設計

> Pro ユーザー限定の月 1 オンラインミーティング（Zoom）の参加受付フォーム。
> Phase 1（2026-04-18〜）のアプリ統合前の MVP 軽量運用。
> アプリの `/events` ページからこのフォームに誘導する。

---

## 📋 質問項目（全 5 問）

### Q1. お名前（ニックネーム可）
- 形式: **記述式（短文）**
- 必須: Yes
- 説明文:
  > Rizup で使っているニックネームで OK です。

### Q2. Rizup のアカウント（メールアドレス or プロフィール名）
- 形式: **記述式（短文）**
- 必須: Yes
- 説明文:
  > Pro プラン確認のため、登録メールアドレスまたはアプリ内のニックネームを記入してください。

### Q3. Pro プランに加入していますか？
- 形式: **ラジオボタン**
- 必須: Yes
- 選択肢:
  - [ ] はい（Pro に加入中）
  - [ ] いいえ（Free プラン）← その場合「ご案内できる枠が Pro 優先のため、今回は見送らせていただきます」と表示するセクション分岐を設定
- セクション分岐:
  - 「はい」→ Q4 へ
  - 「いいえ」→ 専用ページへ（Pro 案内 + 設定画面リンク）

### Q4. 今月話したいこと・聞きたいこと（自由記述）
- 形式: **段落（長文）**
- 必須: No（聞き専 OK のため任意）
- 説明文:
  > 参加者シェアタイム（後半 45 分）で話したいテーマや、翔平さんに聞いてみたいこと。
  > 書かなくても OK。聞き専でも大歓迎です。
- 例表示:
  > 例: 朝活が 3 日坊主で続かない / 副業のはじめ方 / 転職活動の進め方 / 起業の体力管理

### Q5. 参加希望回
- 形式: **ラジオボタン**
- 必須: Yes
- 選択肢（毎月更新）:
  - [ ] 第 1 回：〇月〇日（土）20:00〜21:30
  - [ ] 次回以降の案内を希望
- 説明文:
  > 参加確定者には、開催 2 日前までに Zoom リンクをメールでお送りします。

## ⚙️ フォーム設定

| 項目 | 設定値 |
|---|---|
| フォームタイトル | Rizup 月 1 オンラインミーティング 参加フォーム |
| 説明 | Pro 会員限定の Zoom ミーティング。翔平さんの近況 45 分 + 参加者シェアタイム 45 分。 |
| 回答確認メール | ON（自動送信）|
| 回答後のメッセージ | 「参加登録ありがとうございます。開催 2 日前までに Zoom リンクをお送りします。」 |
| スプレッドシート連携 | ON（回答を一覧管理用）|

## 🔗 フォーム URL 運用

Google フォーム公開後、環境変数 `NEXT_PUBLIC_EVENTS_FORM_URL` に設定する:

1. Google Forms で作成後、「送信」ボタンから共有 URL をコピー
2. Vercel → Project → Settings → Environment Variables
3. `NEXT_PUBLIC_EVENTS_FORM_URL` = `https://forms.gle/XXXXXXXX`
4. 再デプロイでアプリ `/events` ページのボタンから遷移可能に

---

## 📮 フォーム回答後のフロー

### 翔平さん側（運営）
1. フォーム回答が入るたびに Gmail に通知メール
2. スプレッドシートで参加者を一覧管理
3. 開催 2 日前に Zoom リンクを BCC でまとめて送信
4. 欠席連絡用にリマインド時に「参加できなくなった方はこちらから連絡」リンク入れる

### ユーザー側
1. フォーム送信 → 自動返信メール受信（「登録ありがとう」）
2. 開催 2 日前 → Zoom リンクメール受信
3. 当日 → Zoom に入る

## 🔮 Phase 2 移行時（アプリ統合後）

Phase 2 で `events` テーブルをアプリに実装したら、Google フォームは廃止予定。
移行タイミングの判断基準:
- Pro ユーザーが 30 人以上
- 月 1 ミーティングが 3 回連続で定例化
- スプレッドシート管理が手間になってきた

移行時にやること:
- Supabase `events` + `event_attendees` テーブル作成
- アプリ `/events` ページに「参加する」ボタン機能実装
- Google フォームは「アプリからの登録をお願いします」メッセージを表示して閉じる
