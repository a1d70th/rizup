## 各画面の機能（シンプルに）

### `/home`（200行以内・1本フィード）
- ステータスバー1行：朝夜チェック・🔄習慣 n/m・🔥連続
- タイムライン：単一フィード（タブ削除済）
- pull-to-refresh／更新ボタン／＋投稿リンク
- pagination は撤去済（500件一括）

### `/journal`（朝夜同画面・投稿ボタン固定）
- 朝：気分5段階・睡眠時間・🎯今日の一言目標（強調）・本文（任意）・ToDo 3つまで
- 夜：気分・睡眠・朝目標達成（yes/partial/no）・ToDo達成・感謝3つ・本文
- 投稿ボタンは `fixed bottom-16`（常時押下可）

### `/vision`
- 4階層（最終/3年/1年/今月）
- 進捗は habits/todos から自動計算
- 🤖 AIアドバイス（vision-feedback API）
- アンチビジョンタブ：ビジョンがあれば AI 自動生成、なければ導線表示

### `/habits`
- 最大10個、アイコン付き、ビジョン紐付け
- その日のチェック → habit_logs

### `/growth`
- 複利曲線（理想 vs 実績）
- 気分・睡眠・ポジティブ度のスパークライン
- 30日/90日/全期間切替
- Premium で PDF レポート

### `/profile`
- アバター・名前・Rizupタイプ・星座
- 統計3つ（投稿・連続・応援）
- 複利成長カード（CompoundBar）
- 投稿履歴10件

## 削除した機能と理由

| 機能 | 削除理由 |
|---|---|
| `/today` | 朝ジャーナルで代替（v4.3 で削除済） |
| タイムライン 4タブ | 選択肢＝摩擦。1本フィード化（v4.3 で削除済） |
| アンチビジョン手動追加 | AI自動生成で十分（v4.3 で削除済） |
| 紙吹雪 Confetti | 演出過多。連続数の静かな増加で十分（v4.3.2 で削除） |
| ステータスバーの ToDo 表示 | 朝ジャーナル内で完結（v4.3 で削除済） |
| `/recommend` | 毎日のコアループ外。v5 で復活（v4.3.2 で削除） |
| `/api/sho-insight` | フロントから未使用（v4.3.2 で削除） |
| `/anti-vision` 単独ページ | `/vision` タブに統合済（v4.3.2 で削除） |
