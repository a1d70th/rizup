# 事業アイデア置き場

本格事業化前のプランニング・モックアップを保管する場所。
**ここにあるもの ≠ 走ってる事業**。翔平さんが頭の中で検討中のアイデア。

## 昇格フロー

```
company/ideas/*  ← アイデア段階（今ここ）
    ↓ バリデーション通過（MVP 価値あり判定）
    ↓ 1 事業フォルダとして独立
app/{business-name}/  ← 走ってる事業
```

|---|---|---|---|

## 判定基準（このフォルダから昇格する条件）

1. **法的リスクがクリアされている**（ToS 違反 / 個人情報 / 商標衝突 なし）
2. **MVP が 4 週間以内に作れる**
3. **最初の 10 人顧客 or 1,000 PV の獲得経路が具体的**
4. **Rizup の開発リソースを削らないで済む**（Rizup 本業が止まると本末転倒）

4 つ全部クリアした時だけ `app/` に昇格。
