## 議題1: アイコン・ロゴ「Rがかっこよくない」

### 現状
- logo-r.svg: ミント背景 `#6ecbb0` に白抜きセリフ体の「R」
- Header: 「RIZUP」テキストSVG（太字ゴシック）
- PWA全アイコン: この「R」から機械生成

### 問題点
- セリフ体の「R」がウェルネスアプリの世界観（やさしさ・あたたかみ）と合わない
- Finch=鳥、Duolingo=フクロウ。文字だけのアイコンはApp Storeで埋もれる

### 提案
**ミント背景に白い卵キャラ（目とほっぺ付き）** に差し替え
- アプリの育成コンセプトが一目で伝わる
- 小サイズ（16px）でも認識可能なシンプルさ
- gen-icons.mjs で全サイズ再生成するだけ

## 議題2: キャラ育成「たまごからすぐ生まれたら意味わからない」

### 現状（stageOf関数）
```
streak 0     → 卵（実質0日）
streak 1-3   → 赤ちゃん（1日書いただけで即孵化）
```
**→ 卵期間が実質ゼロ。「あたためている感」がない**

### Finchの参考事例
卵期間3-4日。毎日違うメッセージ：
- 1日目「あたためてくれてありがとう」
- 2日目「中で動いてる気がする」
- 3日目「もうすぐ会えるよ...！」→ 孵化

### 提案：卵期間を3日に延長

```
streak 0-2   → 卵（3日間あたためる）
  streak 0: きれいな卵「今日書くとあたため始めるよ🥚」
  streak 1: 小さなヒビ1本「あたたかい。中で何か動いてる」
  streak 2: ヒビ複数+揺れ「もうすぐ会えるよ...！」
streak 3-6   → 赤ちゃん（4日間）→ 孵化アニメーション！
streak 7-20  → 子供（14日）
streak 21-44 → 大人（24日）
streak 45-74 → 村人（30日）
streak 75-99 → 村長（25日）
streak 100+  → 伝説
```

**この「3日間のあたため期間」が初期離脱を防ぐカギ**

## 議題3: みんな（村）ページ

### 現状の問題
1. 友達追加のAPIが未実装（村に人が増える手段がない）
2. 1人で使うと常に空っぽ
3. 「みんな」というタブ名が曖昧

### 提案：1人でも楽しい「森」に

1. **タブ名変更**: 「みんな」→「森」
2. **箱庭化**: streakに応じて背景が変化
   - streak 3: 花が咲く
   - streak 7: 木が生える
   - streak 14: 小川が流れる
   - streak 30: 家が大きくなる
3. **NPC村人**: 「森の精」的なBotキャラを1体配置。日替わりメッセージ
4. **友達機能は中期課題**: まず1人でも楽しい設計にして、ソーシャルは後から追加

## 議題4: 動物が「かわいくない」

### 原因分析
| 問題 | 現在値 | かわいいキャラの基準 |
|------|--------|---------------------|
| 目のサイズ | 顔の23% | 30-40%（Finch: 40%）|
| 頭身 | 1:1 | 2.5:1（頭でっかち）|
| 色の彩度 | くすみ系 | パステルで彩度高め |
| ほっぺ | opacity 0.5 | 0.7-0.8 |
| ハイライト | r=2（小さい） | r=4 + 第2ハイライト |

### 提案：即時改善（SVGパラメータ変更のみ）
1. **目を1.5倍**: `r="8"` → `r="11"`
2. **ハイライト強化**: `r="2"` → `r="4"` + 第2ハイライト `r="1.5"` 追加
3. **体を縮小**: `ry="35"` → `ry="28"`（頭を相対的に大きく）
4. **ほっぺ濃く**: opacity `0.5` → `0.7`、サイズ1.3倍
5. **パレット彩度UP**: 各動物の body を 1段階明るく
