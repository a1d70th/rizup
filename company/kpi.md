# 30万円達成までの各社員 KPI

> **全社目標：月間売上 ¥300,000 達成**
> **達成期限：3ヶ月以内**

---

## 収益構成と各社員の売上責任

```
月30万円の内訳
├── Leo（営業）     ¥150,000  ← クラウドワークス案件
├── Rei（マーケ）   ¥ 80,000  ← note + アフィリエイト
```

## 各社員の KPI 詳細

### 🤖 Haru（秘書・PM）

Haru は直接売上を持たないが、全社員の生産性に責任を持つ。

| KPI | 指標 | 月間目標 |
|---|---|---|
| タスク完了率 | 割り振ったタスクのうち期限内に完了した割合 | 90%以上 |
| 日報作成率 | 営業日のうち日報を書いた日の割合 | 100% |
| Notion 更新頻度 | タスクボード・KPIの更新回数 | 毎日1回以上 |
| ボトルネック検知 | 遅延タスクを24時間以内に検知・報告 | 100% |

**Haru が追う全社ダッシュボード：**

```
今月の進捗
├── 売上実績        ¥_____ / ¥300,000（達成率 ___%）
├── 案件受注数      ___件 / 目標 ___件
├── note 記事公開数 ___本 / 目標 ___本
├── X フォロワー    ___人 / 目標 ___人
└── LP PV          ___PV / 目標 ___PV
```
