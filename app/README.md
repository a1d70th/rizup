# アプリ開発事業部

> 「前向きになれる」体験をプロダクトとして届ける事業部。Rizup の長期的な成長エンジン。

---

## ミッション

コミュニティで培った「引き上げる・前向きになれる」体験を、アプリというプロダクトに昇華させ、より多くの人に届ける。

---

## プロダクト構想

### Phase 1：Rizup App（MVP）

**コンセプト：** 毎日の小さな一歩を記録・共有して、仲間と前向きになれるアプリ

**主要機能**

| 機能 | 内容 |
|---|---|
| 今日の気分チェックイン | 3択で今日の気分を記録。継続するとグラフで可視化 |
| 小さな目標宣言 | 今日やること1つを宣言。達成したらお祝いアニメーション |
| みんなの投稿タイムライン | コミュニティメンバーの一言を見て、共感・応援できる |
| AIからの一言 | 気分に合わせたAIからのメッセージ（励まし・ヒント） |
| 週間振り返り | 今週の記録をサマリーで表示。小さな変化を可視化 |

**技術スタック（予定）**

| カテゴリ | 技術 |
|---|---|
| フロントエンド | React Native（iOS/Android 対応） |
| バックエンド | Supabase（認証・DB・リアルタイム） |
| AI機能 | Claude API |
| デザイン | Figma |
| インフラ | Vercel / Expo EAS |

## 開発ロードマップ

```
Phase 1（〜3ヶ月）  ← 今ここ
  └ MVP設計・ワイヤーフレーム作成
  └ 主要機能の開発（チェックイン・目標宣言）
  └ コミュニティメンバー向けβテスト

Phase 2（3〜6ヶ月）
  └ フィードバックをもとに改善
  └ AI機能（励ましメッセージ）の実装
  └ App Store / Google Play にリリース

Phase 3（6ヶ月〜）
  └ 有料プラン（月額 480円〜）の導入
  └ メンターとのマッチング機能
  └ 企業向けウェルネスプランの展開
```
