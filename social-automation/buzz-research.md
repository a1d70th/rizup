# バズ投稿リサーチ（毎週月曜・Kaiが実施）

> 作成：Kai（CMO）/ 2026-04-13

---

## 🎯 目的

Threads / X でバズっている **自己成長・習慣化・起業系** の投稿パターンを分析し、
Rizupの翔平さんの言葉に**置き換えて再発信**することで、0からリーチを伸ばす。

---

## 🕐 実施タイミング

**毎週月曜 09:00〜10:00**（1時間）

---

## 🔍 リサーチ手順

### 1. プラットフォーム別 TOP10 抽出

#### Threads
- 検索：`自己成長` `習慣` `起業` `モーニングルーティン` `複利` `朝活`
- フィルタ：過去7日間・いいね1,000以上
- 上位10件を URL・いいね数・投稿文とともに記録

#### X
- 同じキーワードで Explore検索
- いいね10,000以上でフィルタ
- 上位10件を記録

### 2. パターン分析テーブル

`buzz-log-YYYY-MM-DD.md` として保存：

| # | URL | 投稿者 | いいね | パターン | 要素 |
|---|---|---|---|---|---|
| 1 | threads.net/... | @xxx | 12,300 | 数字提示 | 具体数値 / Before-After / 短文 |
| 2 | x.com/... | @yyy | 24,500 | 共感 | 感情語 / 疑問文 / 箇条書き |
| 3 | ... | | | | |

### 3. バズパターンの分類

過去の分析で判明している **7つのバズパターン**：

| # | パターン名 | 特徴 | 例 |
|---|---|---|---|
| A | **数字の衝撃** | 「3ヶ月で〇〇%改善」 | ROAS 78%→180% |
| B | **弱さ開示** | 「昨日3時間迷ってた」 | 完璧じゃない人間味 |
| C | **対比構造** | 「× / ○」で比較 | × ありがとう / ○ 具体的褒め |
| D | **問いかけ** | 「〜？」で終わる | 「あなたの今日の1%は？」 |
| E | **実体験ストーリー** | 400名・3ヶ月など具体時間 | GU店長時代の話 |
| F | **定義の言い換え** | 常識の反転 | 「複利は見えないから続かない」 |
| G | **決意表明** | 数字＋期限 | 「来週CW20件送る」 |

## 🎨 Rizup版へのアレンジルール

### 絶対守る3原則

1. **翔平さんの実体験に置き換える**
   - 他人のエピソードをそのまま使わない
   - GU / ROAS / TOEIC / 無職1ヶ月 / Claude Code のどれかに絡める

2. **Rizupのコンセプトに紐付ける**
   - 複利 / 習慣 / ビジョン / 逆算 / 応援のいずれか
   - 最後に `→ https://rizup-app.vercel.app` or `#Rizup`

3. **数字を翔平さんの実績に変える**
   - ✅「離職率28%→1.4%」「ROAS 78%→180%」「TOEIC 850」「月商54万円」
   - ❌ 知らない数字をでっち上げない

### アレンジ例

**元バズ投稿**：
```
年収1000万のコンサルタントが言ってた。
「成功者ほど朝5時起き」
逆だった。
朝起きれるやつが成功する。
```

**Rizup版**：
```
GU時代、400名の部下を見てきて気づいた：

× 売上トップの店員は努力家だった
○ 毎日同じ時間に店に着く店員が、いつの間にかトップになってた

朝の時間を固定する人が勝つ。
意思力じゃなくて、仕組みで動く。

Rizupの習慣トラッカーを作ったのはこの経験から。
```

## 🚨 NGルール（これだけは絶対やらない）

- ❌ 他人の投稿をそのままコピペ（盗用・炎上リスク）
- ❌ 根拠のない数字（「1万人が実践」など検証できないもの）
- ❌ 煽り（「知らない人は損してる」系）
- ❌ Rizupと無関係のバズネタ（一貫性を失う）
- ❌ 数字のねつ造（必ず翔平さんの実績 or 公式ソース）
