## 3. 条件交渉時の切り返しテンプレ

### 「時給を少し下げられないか」と言われた場合

```
ご提案ありがとうございます。

時給につきましては、初月のみ 1,500円 にてご調整可能です。
（2ヶ月目以降、成果を見ていただいた上で 1,700円 に戻させてください）

初月で必ずお約束するもの：
 ① 目標KPIの ●●% 達成（数値で合意）
 ② 週次レポートで進捗を毎週金曜に送付
 ③ 達成できない場合、未達分を翌月稼働で補填

「投資対効果」を最初の30日で明確にお見せします。

いかがでしょうか。
```

まずは今月、週10時間で「質」と「数字」を証明させてください。
```

```
承知いたしました。比較検討、大切です。

もし判断材料になれば、以下を24時間以内にお送りできます：
 ・過去のROAS改善レポート（匿名化済みサンプル）
