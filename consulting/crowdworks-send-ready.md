## 📋 送信時の手順

### 1. クラウドワークスにログイン
https://crowdworks.jp/login

### 2. スカウトメッセージを開く
- トップ → 右上メニュー → **「スカウト一覧」**
- 対象のスカウトをクリック

### 3. 返信フォームに貼り付け
- 件名：上記の「📧 件名」をコピペ
- 本文：上記の「📝 本文」をコピペ
- `○○様` を実際の担当者名に置換（3箇所）

### 4. 送信前プレビュー確認
- 改行が崩れていないか
- `━━` の罫線が揃っているか
- 空白・誤字がないか

### 5. 送信
**「送信する」** をクリック。

### 6. 記録
`consulting/pipeline.md` に追記：

```markdown
## 2026-04-14 スカウト① 条件提示送信
- 送信日時：2026-04-14 09:XX
- 先方：XXX株式会社 ○○様
- 提示条件：時給1,700円・週10h
- Zoom候補：4/14, 4/15, 4/16 各3枠
- 返信期待日：48時間以内
```

## 💬 返事の内容別：切り返し早見表

### 返答パターン1：「条件OK、契約進めたい」
→ クラウドワークスで「契約書」ボタン → Zoom 日程確定 → キックオフ資料準備

### 返答パターン2：「もう少し時給を下げられないか」
→ 以下を返信：
```
ご提案ありがとうございます。

時給については、初月に限り 1,500円 まで調整可能です。
（2ヶ月目以降、成果を見ていただいた上で 1,700円 に戻させてください）

初月でお約束：
① 目標KPIの ●●% 達成
② 週次レポートで進捗を数値可視化
③ 達成できなければ翌月稼働で補填

いかがでしょうか。
```

### 返答パターン3：「もっと多く稼働してほしい」
→ 以下を返信：
```
お声がけありがとうございます。

並行案件の調整上、今月は 週10時間 が
確実にコミットできる上限となります。

5月以降であれば 週15〜20時間 に拡大可能です。
ご希望あればお気軽にお申し付けください。

まずは今月、週10時間で「質」と「数字」を証明させてください。
```

### 返答パターン4：「他の候補者と比較したい」
→ 以下を返信：
```
承知いたしました。比較検討は大切です。

判断材料として、以下を24時間以内にお送りできます：
・過去のROAS改善レポート（匿名化済）
・類似案件の月次KPI推移
・導入前後の Before/After 各30日分の数字比較

ご希望でしたらお申し付けください。
```
