## 📋 送信時の手順

### 1. クラウドワークスにログイン
https://crowdworks.jp/login

### 2. スカウト一覧を開く
トップ → 右上メニュー → **「スカウト一覧」** → 対象スカウトをクリック

### 3. 返信フォームに貼り付け
- 件名：上記 A/B/C から選択してコピペ
- 本文：上記「📝 本文」をコピペ
- `○○` を実際の名前に置換（Ctrl+F で全件置換）

### 4. プレビュー確認 → 送信

### 5. 送信後記録（`consulting/pipeline.md`）
```markdown
## 2026-04-16 スカウト① 条件提示送信
- 送信日時：2026-04-16 XX:XX
- 先方：XXX株式会社 ○○様
- 使用件名：A / B / C のどれか
- 提示条件：時給1,700円・週10h・月68,000円〜
- Zoom候補：4/17, 4/20, 4/21 各3枠
- 返信期待日：48時間以内
```
