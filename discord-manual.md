## 1. チャンネル構成

### カテゴリ構成の全体像

```
Rizup サーバー
│
├── 📋 スタートここから
│   ├── 📌 rules（ルール・約束ごと）
│   ├── 👋 welcome（ウェルカムメッセージ・自動送信）
│   └── 📣 announcements（お知らせ）
│
├── 🌱 はじめましょう
│   ├── 🙋 self-intro（自己紹介）
│   └── 🎯 my-goals（今月の目標宣言）
│
├── 💬 話しましょう
│   ├── 🌤 daily-checkin（今日の一言・気分報告）
│   ├── 😔 share-worries（悩み・モヤモヤを話す）
│   ├── 🎉 share-wins（小さな達成を報告する）
│   └── 💡 tips-and-ideas（気づきや学びをシェア）
│
├── 🤝 サポート
│   ├── 🔰 ask-anything（なんでも質問）
│   └── 📚 resources（おすすめ記事・ツール）
│
└── 🏠 運営
    ├── 📝 feedback（サーバーへの要望・感想）
    └── 🔒 admin-log（運営メモ・非公開）
```

### 各チャンネルの役割と運用メモ

| チャンネル | 目的 | 運用のポイント |
|---|---|---|
| `rules` | サーバーの約束ごとを明示 | ピン留め必須。入室時に必ず読んでもらう |
| `welcome` | 参加者への自動歓迎 | MEE6 や Carl-bot で自動送信設定 |
| `announcements` | 重要なお知らせ | @everyone メンションはここだけに限定 |
| `self-intro` | 自己紹介の場 | テンプレートをピン留めしておく |
| `my-goals` | 月次の目標宣言 | 月初にリマインドを送る |
| `daily-checkin` | 毎日の気分・近況報告 | 運営が毎日一番に投稿して文化を作る |
| `share-worries` | 悩みを打ち明ける場 | 反応は共感ファーストで。解決策は求められてから |
| `share-wins` | 達成報告・褒め合う場 | どんな小さなことも全力で反応する |
| `tips-and-ideas` | 気づき・学びのシェア | 質より量。短い気づきでも歓迎 |
| `ask-anything` | 質問・相談 | 24時間以内に必ず誰かが反応する |
| `resources` | お役立ち情報 | 運営がキュレーションして定期投稿 |
| `feedback` | サーバー改善の要望 | 月1回まとめて見直す |

## 2. ウェルカムメッセージ

### 自動ウェルカムメッセージ（Bot送信）

参加直後に DM またはチャンネルで自動送信するメッセージ。

```
🌿 Rizup へようこそ！

はじめまして、Rizup 運営です。
参加してくれてありがとうございます。

Rizup は「活躍できていない」と感じている人たちが
お互いを引き上げ、前向きになれる場所です。

---

✅ まず最初にやってほしいこと3つ

1. 📋 #rules を読む（2分でOK）
2. 🙋 #self-intro に自己紹介を投稿する
3. 🌤 #daily-checkin で「今日の気分」を一言書く

---

💬 自己紹介テンプレ（コピーしてどうぞ）

---
📛 なまえ（ニックネームOK）：
🌱 今の状況（一言で）：
😔 最近のモヤモヤ：
🎯 Rizup でやってみたいこと：
🍀 ひとこと：
---

完璧じゃなくていいです。一言でもOK。
あなたのペースで、ゆっくり馴染んでいってください。

何か困ったことがあれば、いつでも声をかけてください 👋
```

---

### `#rules` チャンネルに固定するルール文

```
# Rizup のルール・約束ごと

Rizup は「比べない・批判しない・一緒に前を向く」
そういう場所にしたいと思っています。

---

## 大切にしてほしいこと

✅ 相手の言葉を尊重する
✅ 共感ファーストで反応する（まず「わかるよ」から）
✅ 自分のペースで参加する（毎日来なくてもOK）
✅ 小さな進歩を喜び合う

## やめてほしいこと

❌ 他のメンバーへの批判・否定
❌ 勧誘・宣伝・スパム行為
❌ 個人情報（本名・住所など）の無断共有
❌ 政治・宗教に関する議論

---

ルールに違反する行為があった場合は、
運営の判断でサーバーから退出いただく場合があります。

一緒に、いいコミュニティを作っていきましょう 🌿
```

## 3. 最初の10人の集め方

### 基本の考え方

> 最初の10人は「広める」より「選ぶ」。
> 文化を作ってくれる人を丁寧に口コミで集める。

---

### フェーズ別アクション

#### フェーズ1：0 → 3人（信頼できる身近な人から）

**目的：**「1人でも投稿される場所」を作る。最初の仲間が空気を作る。

- [ ] 自分が「この人なら Rizup に合う」と思う人に直接 DM で招待する
- [ ] 「テスト的に入ってほしい、感想を教えてほしい」という形でお願いする
- [ ] 入ってもらったらその場で自己紹介・リアクションをして温かい空気を作る

**DM テンプレ（知人向け）**
```
こんにちは！最近、悩みを持ちつつも前向きになれる場所を作りたくて、
小さなコミュニティを立ち上げました。

「活躍できていない人を引き上げる・前向きになれる場所」をコンセプトにした
Discord コミュニティです。

まだ立ち上げたばかりで10名限定のβ版なんですが、
〇〇さんに最初のメンバーとして入ってほしいなと思って連絡しました。

使ってみての感想や改善意見をもらえたら嬉しいです。
もし興味あれば、こちらから入れます → https://discord.gg/ssA69BTe4
```

---

#### フェーズ2：3 → 7人（SNS で小さく発信）

**目的：**「一緒に育てたい人」を引き寄せる。数より質。

**X（Twitter）投稿例**

```
こんな気持ちになったことはありますか？

・頑張っているのに、うまくいかない
・自信がなくて、一歩が踏み出せない
・周りと比べて、落ち込んでしまう

そういう人が集まって、一緒に前を向く
小さなコミュニティを作っています。

今だけ無料・10名限定のβ版です。
気になる人はリプか DM ください🌿

#コミュニティ #Discord #自己成長
```

```
「活躍できていない」は、あなたのせいじゃない。

環境・タイミング・比較——
うまくいかない理由はたくさんあるけれど、
一人で抱えると余計につらくなる。

Rizup は、そういうモヤモヤを持ち寄って
一緒に前を向くための場所です。

完全無料・β版・10名限定で始めています🌱
→ https://discord.gg/ssA69BTe4
```

**ポイント**
- フォロワー数より、共感してくれる人との対話を優先
- リプや「わかります」のコメントには必ず返信して関係を作る
- 一度に大量に告知しない（1〜2日に1投稿のペースで続ける）

---

#### フェーズ3：7 → 10人（コミュニティ内の口コミ）

**目的：**既存メンバーが「ここいいよ」と自然に紹介したくなる場を作る。

- [ ] メンバーが「誰かに紹介したくなるような体験」を意識して作る
  - 投稿に丁寧に反応する
  - 名前を呼んで「〇〇さん、ありがとう」を言う
  - 小さな達成を盛大に祝う
- [ ] 「知り合いで悩んでいる人がいたら招待してもいいですよ」と伝える
- [ ] LPのリンクをメンバーがシェアしやすいようにテキストで渡す

**シェア促進テキスト（メンバーへ）**
```
Rizup に合いそうな人がいたら、
ぜひ紹介してもらえると嬉しいです 🙌

こちらのリンクをシェアしてください👇
https://discord.gg/ssA69BTe4

「悩みを一人で抱えてそうな人」「最近元気なさそうな人」——
そういう人こそ、ここに来てほしいです。
```

---

### 10人集めるための注意点

| やること | やらないこと |
|---|---|
| 1人1人に丁寧に向き合う | フォロワーへの一斉 DM 送信 |
| 「来てよかった」体験を作る | 人数集めを目的にする |
| 断られても感謝して終わる | しつこく誘う |
| 毎日自分が投稿して場を温める | メンバーの投稿を待つだけにする |

## 4. 毎日の運営ルーティン

### 所要時間の目安：**1日15〜30分**

---

### 朝のルーティン（10〜15分）

#### 1. `#daily-checkin` に「今日の一言」を投稿する（毎日・必須）

> 運営が毎日投稿することで、「ここは毎日動いている場所」という文化が生まれます。

**投稿テンプレ**
```
おはようございます🌤

今日の気分はどうですか？
一言だけでも教えてもらえると嬉しいです。

（運営の一言：今日は〇〇な気分です。△△しながらスタートしました。）

今日も、自分のペースで行きましょう 🌿
```

**曜日別バリエーション**

| 曜日 | テーマ |
|---|---|
| 月 | 「今週やってみたいこと」 |
| 火 | 「最近気づいたこと・学んだこと」 |
| 水 | 「今の正直な気持ち」 |
| 木 | 「誰かに感謝したいこと」 |
| 金 | 「今週の小さな進歩を振り返る」 |
| 土 | 「休日の過ごし方・リフレッシュ法」 |
| 日 | 「来週やりたいことを一つ宣言」 |

---

#### 2. 前日の投稿を確認してリアクション・返信する（毎日）

- 未返信の投稿がないかチェック
- リアクション（絵文字スタンプ）だけでも OK。「見てるよ」が伝わればいい
- 特に `#share-worries` は24時間以内に必ず反応する

---

### 週次ルーティン

#### 月曜：目標宣言を促す

```
新しい週が始まりました！
#my-goals に「今週やること」を一つ書いてみませんか？

大きなことじゃなくてOKです。
「毎朝水を飲む」「5分だけ本を読む」——そんな小さなことでいいです🌱
```

#### 金曜：振り返りを促す

```
今週もお疲れさまでした 🌿

#share-wins に「今週の小さな進歩」を書いてみてください。
「ちゃんと寝られた」「一回だけ運動した」——それで十分です。

あなたの進歩を、みんなで一緒に喜びたいです🎉
```

#### 週末：コミュニティの健全性チェック（5分）

- [ ] 1週間の総投稿数を確認（減っていたら翌週に活性化施策を考える）
- [ ] 長期間発言がないメンバーに DM でそっと声をかける
- [ ] ルール違反や雰囲気を壊す投稿がないか確認

---

### 月次ルーティン

#### 月初：目標宣言月
- `#my-goals` に今月の目標を書くよう促す
- 先月の達成者を称えるメッセージを投稿する

#### 月末：振り返り月
- 「今月の Rizup まとめ」を `#announcements` に投稿
  - 総投稿数
  - 印象に残った言葉や出来事（匿名でOK）
  - 来月楽しみなこと

#### `#feedback` の確認と反映
- 月1回、要望や感想をまとめて読む
- 対応できたことは「対応しました」と報告する

---

### 運営マインドセット

#### 大切にする3つのこと

1. **毎日自分が一番楽しむ**
   コミュニティの雰囲気は運営の気持ちが伝わる。義務感より楽しさを優先する。

2. **メンバーの名前を呼ぶ**
   「〇〇さん、それすごいですね！」と名前を呼ぶだけで親近感が大きく変わる。

3. **沈黙を怖がらない**
   投稿が少ない日があっても大丈夫。毎日少しずつ積み上げることが大切。

#### 燃え尽きないために

- 1日に使う時間の上限を決める（目安：30分）
- 「全員を助けなければ」と思わない。いるだけで場が温かくなる
- 運営自身がしんどいときは「今日は元気ないです」と正直に投稿していい
