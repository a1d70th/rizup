## 3. ウェルカムメッセージ

### #verify チャンネルに固定するメッセージ

```
🌿 Rizup へようこそ！

このサーバーに参加するには、以下のルールに同意してください。

📌 ルール（3つだけ）
1. 他のメンバーを批判・否定しない
2. 勧誘・スパム・宣伝をしない
3. 自分のペースで参加する（毎日来なくてもOK）

✅ 同意する場合は、下の ✅ リアクションを押してください。
→ 自動的にサーバー内のチャンネルが見えるようになります。
```

> Carl-bot のリアクションロール機能で ✅ を押すと `@メンバー` ロールを自動付与する。

```

参加してくれてありがとうございます。

まず最初にやってほしいこと：
1. 👋 #self-intro に自己紹介を投稿する
2. 🌤 #daily-checkin で今日の気分を一言書く

自己紹介テンプレ（コピーしてどうぞ）：

📛 なまえ：
🌱 今の状況（一言）：
😔 最近のモヤモヤ：
🎯 やってみたいこと：

完璧じゃなくていいです。一言でもOK。
あなたのペースで、ゆっくり馴染んでいってください 🌿
```

## 4. ルール（#rules に固定）

```
# Rizup コミュニティルール

Rizup は「比べない・批判しない・一緒に前を向く」場所です。

## 大切にしてほしいこと

✅ 相手の言葉を尊重する
✅ 共感ファーストで反応する（まず「わかるよ」から）
✅ 自分のペースで参加する（毎日来なくてもOK）
✅ 小さな進歩を喜び合う
✅ 困ったら #ask-anything で気軽に聞く

## やめてほしいこと

❌ 他のメンバーへの批判・否定・マウント
❌ 勧誘・宣伝・スパム行為
❌ 個人情報（本名・住所・電話番号）の無断共有
❌ 政治・宗教・差別に関する議論
❌ 有料エリアの内容を外部に転載

## 違反した場合

1回目：運営からDMで注意
2回目：一時的なミュート
3回目：サーバーから退出

一緒に、いいコミュニティを作っていきましょう 🌿
```

## 5. Bot 設定手順

### 5-1. Carl-bot（リアクションロール・自動ロール）

**用途：** ルール同意でロール付与、定期投稿

**セットアップ手順：**

1. `https://carl.gg` にアクセス → 「Login」→ Discord でログイン
2. Rizup サーバーを選択 → Bot を招待
3. ダッシュボードで以下を設定

**リアクションロール設定：**
```
Dashboard → Reaction Roles → Create new
  Channel: #verify
  Message: 上記のウェルカムメッセージを貼り付け
  Emoji: ✅ → Role: @メンバー
  Mode: Verification（一度押したら取り消し不可）
```

**自動ロール設定：**
```
Dashboard → Automod → Autoroles
  → なし（verifyを通してから付与するため）
```

---

### 5-2. MEE6（ウェルカムDM・レベリング）

**用途：** 参加時の自動DM、アクティブ度の可視化

**セットアップ手順：**

1. `https://mee6.xyz` にアクセス → Discord でログイン
2. Rizup サーバーを選択 → Bot を招待
3. ダッシュボードで以下を設定

**ウェルカムDM：**
```
Dashboard → Welcome → Enable
  Channel: DM（Direct Message）
  Message: 上記の Bot DM ウェルカムメッセージを設定
```

**レベリング（任意）：**
```
Dashboard → Levels → Enable
  → レベルアップ時のメッセージをカスタマイズ
  → レベル5で特別ロール付与なども設定可能
```
